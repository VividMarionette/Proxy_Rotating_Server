# proxies.txt

The application reads proxy addresses from a file named `proxies.txt`.

Each proxy should be listed on a new line using the following format:

```text
IP:PORT
```

Example:

```text
27.76.194.0:1080
103.81.19.162:6666
81.31.246.178:1080
174.138.119.88:80
47.91.65.23:3128
103.105.78.217:3125
154.113.209.162:8082
58.235.170.51:5678
185.132.1.221:4145
```

> **Note:** Some proxy servers may become unavailable over time. Network availability depends on the external proxy service and is outside the control of this application.
